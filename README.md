## RabbitESB

a simple ESB base rabbitmq, just a technique DEMO.

## Requirements

 *  rabbitmq    3.5.*
 *  python      2.7.*
 *  twisted     15.*
 *  pika        0.10.*
 *  nose        1.*


## Quick Start

 1. install requirements
 2. python demo_service.py &
 3. python http_proxy.py &
 4. nosetests
